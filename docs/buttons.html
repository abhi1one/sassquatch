Flat Buttons

Primary Buttons

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

Use primary buttons for primary actions.

    <button class="button--primary">Button</button>
    <button class="button--black">Button</button>
    <button class="button--white">Button</button>
    <button class="button--blue">Button</button>

Secondary Buttons

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

Secondary buttons are slightly transparent so they appear to take on a little bit of the background color behind them.
