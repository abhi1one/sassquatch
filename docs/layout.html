Doc Content

Provides padding for content; acts as a vertical content divider.

first content container

second content container

    <div class="doc-box">
        <div class="doc-content">
            <p>first content container</p>
        </div>
        <div class="doc-content">
            <p>second content container</p>
        </div>
    </div>

Full width lists and doc-content

You can add a sectionList directly inside the doc-box in order to to span the full width of the box, and use doc-content for padded content:

Cat name ideas

  • Jennifer

  • Sergei

  • Bruce Springsteen

  • Mittens

    <div class="doc-box">
        <div class="doc-content">
            <h3>Cat name ideas</h3>
        </div>
        <ul class="sectionList">
            <li><p>Jennifer</p></li>
            <li><p>Sergei</p></li>
            <li><p>Bruce Springsteen</p></li>
            <li><p>Mittens</p></li>
        </ul>
    </div>

OOCSS Grids

OOCSS Grids are used to break any space into fractions, they can be nested and stacked. A unit can contain another line or it can contain other objects directly. Grids control width, content controls height.

A .line or .line-gutters is a row, and a .unit is a column.

lastUnit flexes to fit the remaining space in the row.

...first column...

...second column...

...third column...

...first column...

...second column...

...third column...

    <div class="line">
        <div class="unit size1of3">
            <p class="shaded">...first column...</p>
        </div>
        <div class="unit size1of3">
            <p class="highlighted">...second column...</p>
        </div>
        <div class="lastUnit">
            <p class="shaded-red">...third column...</p>
        </div>
    </div>

    <div class="line-gutters">
        <div class="unit size1of3">
            <p class="shaded">...first column...</p>
        </div>
        <div class="unit size1of3">
            <p class="highlighted">...second column...</p>
        </div>
        <div class="lastUnit">
            <p class="shaded-red">...third column...</p>
        </div>
    </div>

A number of size classes are available:

  • size1of1
  • size1of2
  • size1of3, size2of3,
  • size1of4, size3of4
  • size1of5, size2of5, size3of5, size4of5
  • size1of7, size2of7, size3of7, size4of7, size5of7, size6of7
