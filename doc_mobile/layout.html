Stripe Layout

Used for newer and more responsive layouts, where content lives on top of a full-width background instead of docBox modules. The docStripe container sets a full width background.

The docBounds class is used as a padding helper and content wrapper; it uses a max-width and min-width, but will otherwise flex to fit the width of the parent container.

I'm in a docBounds within a stripe

Any arbitrary content can go here. The docBounds sets the margins and max with. The stripe behind it sets the color.

I'm in a red stripe

This docStripe also has a class of inverted to style these child element text colors correctly

Fix-flex

A collection of styles that allow us to do tabular internal layouts on list items, etc. It's particularly useful on responsive layouts, like we use for mobile.

Note

  • There should be only one .flex per row.

  • If you want to set minimum heights on fix, flex or ffbox, you'll need to use height rather than min-height because of the way table layouts are used in CSS.

  • In the future, we'll probably want to swap out tabular layouts with flexbox.

  • If you want to use CSS text truncation with ffbox, you need to use the ffbox--fixed class on the ffbox, then set a class with a width value on the nested ffbox-fix, like so:

fix
flex 
    <div class="ffbox">
        <div class="ffbox-fix shaded-red"> fix </div>
        <div class="ffbox-flex guide-shade"> flex </div>
    </div>

Pin layout

By default, both Mobile's _figureset-m.scss and _fix-flex.scss are middle aligned. The pinning system allows you to append to either the "table" components of a figureset or ffbox OR the "table cell" components to adjust that alignment to the top or bottom.

Class Description
pintop sets vertical alignment to display: tabel-cell; elements to top
pinbottom sets vertical alignment to display: tabel-cell; elements to bottom
fix
flex
fix
flex 
    <!-- Pinning cells -->
    <div class="ffbox" style="min-height: 40px;">
        <div class="pintop ffbox-fix shaded-red"> fix </div>
        <div class="ffbox-flex guide-shade"> flex </div>
    </div>

    <hr />

    <!-- Pinning whole ffbox units-->
    <div class="ffbox pintop" style="min-height:40px;">
        <div class="ffbox-fix shaded-red"> fix </div>
        <div class="ffbox-flex guide-shade"> flex </div>
    </div>

OOCSS Grids

OOCSS Grids are used to break any space into fractions, they can be nested and stacked. A unit can contain another line or it can contain other objects directly. Grids control width, content controls height.

A .line or .line-gutters is a row, and a .unit is a column.

lastUnit flexes to fit the remaining space in the row.

...first column...

...second column...

...third column...

...first column...

...second column...

...third column...

    <div class="line">
        <div class="unit size1of3">
            <p class="shaded">...first column...</p>
        </div>
        <div class="unit size1of3">
            <p class="highlighted">...second column...</p>
        </div>
        <div class="lastUnit">
            <p class="shaded-red">...third column...</p>
        </div>
    </div>

    <div class="line-gutters">
        <div class="unit size1of3">
            <p class="shaded">...first column...</p>
        </div>
        <div class="unit size1of3">
            <p class="highlighted">...second column...</p>
        </div>
        <div class="lastUnit">
            <p class="shaded-red">...third column...</p>
        </div>
    </div>
